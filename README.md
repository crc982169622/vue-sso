# vue-sso

该项目是学习前后端分离（前端vue + 后端springboot）项目的SSO单点登录、单点注销代码

项目目录结构：
    
    --- vue-sso 总项目目录
        --- vue-server1  后台服务1
        --- vue-server2  后台服务2
        --- vue-sso-common  公共服务
        --- vue-sso-front  前端vue项目
        --- vue-sso-redis  redis共享服务
        --- vue-sso-server  sso登录服务
